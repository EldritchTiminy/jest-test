# Jest Practice
Focused study and practice in the use of Jest for automated JavaScript testing.

## Features
Simple JavaScript tasks designed for unit testing via Jest

## Installation
1. Clone the repository
```bash
  git clone https://github.com/EldritchTiminy/jest-test.git
  cd jest-test
```

2. Install dependencies
```bash
  npm install
```

3. Start the developement server
```bash
  npm run dev
```

## Usage
- After setup run:
  ```bash
  npm test
  ```
- Jests automated tests should display in the console.
